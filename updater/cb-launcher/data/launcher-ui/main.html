Welcome to CB Servers Launcher! Select a game from the sidebar to get started.
This launcher provides unified access to all game clients with host servers on:
  • Black Ops 3 - Enhanced with BOIII modifications
  • Ghosts - Enhanced with IW6x modifications
  • Advanced Warfare - Enhanced with S1x modifications
  • Modern Warfare Remastered - Enhanced with H1-Mod features

When enabled, the launcher will automatically open the last game page you were viewing.
When enabled, game file verification will skip hash checks for much faster validation at the cost of file integrity.
When enabled, the launcher will automatically close after launching a game.
